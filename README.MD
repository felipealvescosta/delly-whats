## Getting Started

To get a local copy up and running follow these simple steps.


### Installation
 
1. Clone the repo
```sh
git clone https://github.com/felipealvescosta/delly-whats.git
```
2. Install packages
```sh
yarn install
```
3. Run android project
```sh
yarn android
```
3. Run ios project
```sh
cd ios && pod install
```
```sh
yarn ios
```
